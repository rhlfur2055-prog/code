## [1] print() 기초
📚 print()=화면출력, 괄호안 내용 출력, 자동줄바꿈
📝 `____("Hello")` → print
🎯 함정: 따옴표/괄호 누락
⚠️ print vs Print (대소문자)
🔄 `print("A","B")` → A B

## [2] print() sep/end
📚 sep=값사이구분자(기본공백), end=끝문자(기본줄바꿈)
📝 `print(1,2,____="-")` 출력 1-2-3 → sep
🎯 함정: sep와 end 혼동
⚠️ sep=""이면 붙여서 출력
🔄 `print("A",end="");print("B")` → AB

## [3] input() 기초
📚 input()=키보드입력받기, 항상 문자열 반환, 괄호안=안내문
📝 `name=____("이름:")` → input
🎯 함정: input()은 무조건 str 반환
⚠️ 숫자 입력해도 문자열!
🔄 `x=input(); print(type(x))` → <class 'str'>

## [4] input() 형변환
📚 int(input())=정수변환, float(input())=실수변환
📝 `age=____(input("나이:"))` 정수로 → int
🎯 함정: int 안하면 연산 오류
⚠️ "3"+"5"="35", int("3")+int("5")=8
🔄 `x=int(input()); print(x*2)` 입력3 → 6

## [5] 변수 기초
📚 변수=값저장공간, 이름=값 형식, 재할당 가능
📝 `x __ 10` → =
🎯 함정: =는 할당, ==는 비교
⚠️ 변수명 숫자시작 불가 (1abc X)
🔄 `x=5; x=10; print(x)` → 10

## [6] 변수 명명규칙
📚 영문/숫자/언더스코어, 숫자시작불가, 예약어불가
📝 유효한 변수명? 1abc(X) _abc(O) for(X) my_var(O)
🎯 함정: class, if, for 등 예약어
⚠️ 대소문자 구분 (Name≠name)
🔄 `my_name="Kim"; myName="Lee"` → 둘 다 유효

## [7] int 정수형
📚 정수=소수점없는수, int()로변환, //나눗셈결과도정수
📝 `type(10)` → <class 'int'>
🎯 함정: 10/2=5.0 (float), 10//2=5 (int)
⚠️ int(3.9)=3 (버림!)
🔄 `int("100")` → 100

## [8] float 실수형
📚 실수=소수점있는수, /나눗셈결과, float()변환
📝 `type(3.14)` → <class 'float'>
🎯 함정: 10/5=2.0 (항상 float)
⚠️ float("3.14")=3.14
🔄 `1+2.0` → 3.0 (float으로 변환)

## [9] str 문자열형
📚 문자열=따옴표로감싼텍스트, ""또는''사용
📝 `type("hello")` → <class 'str'>
🎯 함정: "3"과 3은 다름
⚠️ "3"*2="33", 3*2=6
🔄 `str(100)` → "100"

## [10] bool 불린형
📚 불린=True/False, 조건결과, 대문자시작필수
📝 `type(True)` → <class 'bool'>
🎯 함정: true(X) True(O)
⚠️ bool(0)=False, bool(1)=True
🔄 `bool(""), bool("a")` → False, True

## [11] 산술연산자 +-*/
📚 +덧셈, -뺄셈, *곱셈, /나눗셈(항상float)
📝 `10/4` → 2.5
🎯 함정: /결과는 무조건 float
⚠️ 10/2=5.0 (int 아님!)
🔄 `7+3, 7-3, 7*3, 7/3` → 10,4,21,2.333...

## [12] 산술연산자 //, %, **
📚 //몫, %나머지, **거듭제곱
📝 `10//3, 10%3, 2**3` → 3, 1, 8
🎯 함정: /와 //혼동
⚠️ -7//2=-4 (내림!)
🔄 `17%5` → 2

## [13] 비교연산자 ==, !=
📚 ==같다, !=다르다, 결과는 bool
📝 `3__3` True되려면 → ==
🎯 함정: =할당 vs ==비교
⚠️ "3"==3 → False (타입다름)
🔄 `5!=5, 5!=6` → False, True

## [14] 비교연산자 <, >, <=, >=
📚 <작다, >크다, <=이하, >=이상
📝 `5__3` True되려면 → > 또는 >=
🎯 함정: <=와 <혼동
⚠️ 5<=5 → True
🔄 `3<5, 3>5, 3<=3` → True,False,True

## [15] 논리연산자 and
📚 and=둘다True면True, 하나라도False면False
📝 `True and ____` False되려면 → False
🎯 함정: and의 단축평가
⚠️ False and X → X 평가 안함
🔄 `True and True, True and False` → True, False

## [16] 논리연산자 or
📚 or=하나라도True면True, 둘다False면False
📝 `False or ____` True되려면 → True
🎯 함정: or의 단축평가
⚠️ True or X → X 평가 안함
🔄 `False or False, False or True` → False, True

## [17] 논리연산자 not
📚 not=반대로, True→False, False→True
📝 `not True` → False
🎯 함정: not의 우선순위
⚠️ not True and False → False and False → False
🔄 `not False, not 0, not ""` → True, True, True

## [18] 복합대입연산자
📚 +=더하고대입, -=빼고대입, *=, /=, //=, %=
📝 `x=5; x____2; print(x)` 7되려면 → +=
🎯 함정: x+=1과 x=x+1 동일
⚠️ 문자열도 += 가능 ("a"+="b" → "ab")
🔄 `x=10; x-=3; x*=2` → 14

## [19] 연산자 우선순위
📚 **>*,/,//,%>+,->비교>not>and>or
📝 `2+3*4` → 14 (곱셈먼저)
🎯 함정: 괄호로 우선순위 조절
⚠️ (2+3)*4=20, 2+3*4=14
🔄 `2**3**2` → 512 (오른쪽부터)

## [20] 형변환 정리
📚 int()정수, float()실수, str()문자열, bool()불린
📝 `int("10")+int("20")` → 30
🎯 함정: int("3.14") 에러!
⚠️ int(float("3.14"))=3 (우회)
🔄 `str(123)+"원"` → "123원"

## [21] if문 기본
📚 if 조건: (콜론필수), 들여쓰기블록실행
📝 `if x>0__` → :
🎯 함정: 콜론 누락
⚠️ 들여쓰기 4칸 또는 탭
🔄 `if True: print("실행")` → 실행

## [22] if-else문
📚 if 조건: A else: B, 조건False면 else실행
📝 `if x>0: print("양") ____: print("음")` → else
🎯 함정: else에도 콜론
⚠️ else 조건 없음
🔄 `if False: A else: B` → B실행

## [23] if-elif-else문
📚 elif=else if, 여러조건 순차검사
📝 다중조건시 `if-____-else` → elif
🎯 함정: elif 순서 중요
⚠️ elif에도 조건과 콜론 필요
🔄 `if x>90: A elif x>80: B else: C`

## [24] 중첩 if문
📚 if안에 if, 들여쓰기로 구분
📝 내부조건 들여쓰기 → 8칸(2단계)
🎯 함정: 들여쓰기 레벨 혼동
⚠️ 같은레벨=같은블록
🔄 `if x>0: if x>10: print("big")`

## [25] 조건식 in 연산자
📚 in=포함여부, 문자열/리스트에 사용
📝 `"a" __ "apple"` True → in
🎯 함정: in은 대소문자구분
⚠️ "A" in "apple" → False
🔄 `3 in [1,2,3]` → True

## [26] 조건식 not in
📚 not in=미포함여부
📝 `"x" ____ "apple"` True → not in
🎯 함정: not in vs in not
⚠️ in not은 문법오류!
🔄 `5 not in [1,2,3]` → True

## [27] 삼항연산자
📚 A if 조건 else B, 한줄조건문
📝 `"양수" if x>0 ____ "음수"` → else
🎯 함정: if와 else 순서
⚠️ 조건True면 앞, False면 뒤
🔄 `"짝" if n%2==0 else "홀"`

## [28] pass문
📚 pass=아무것도안함, 빈블록채우기용
📝 `if x>0: ____` 아무것도안하려면 → pass
🎯 함정: pass vs continue
⚠️ pass는 그냥통과, continue는 반복건너뜀
🔄 `def f(): pass` (빈함수)

## [29] 여러조건 결합
📚 and/or로 조건결합
📝 `x>0 ____ x<10` 둘다만족 → and
🎯 함정: 괄호로 우선순위 명확히
⚠️ (A or B) and C ≠ A or (B and C)
🔄 `if 0<x<10:` 파이썬특수문법

## [30] 조건문 종합
📚 조건문=분기처리, if/elif/else조합
📝 점수→학점 변환 로직
🎯 함정: elif 순서, 경계값
⚠️ >=90 vs >90 차이
🔄 `if s>=90:A elif s>=80:B else:C`

## [31] for문 기본
📚 for 변수 in 반복가능객체:, 순회
📝 `____ i in [1,2,3]: print(i)` → for
🎯 함정: for 뒤 콜론 필수
⚠️ 변수는 자동할당
🔄 `for x in "abc": print(x)` → a b c

## [32] range() 기본
📚 range(n)=0부터n-1까지, range(a,b)=a부터b-1까지
📝 `range(5)` → 0,1,2,3,4
🎯 함정: range(5)에 5 미포함!
⚠️ range(1,5) → 1,2,3,4
🔄 `for i in range(3): print(i)` → 0 1 2

## [33] range() step
📚 range(a,b,step)=step간격으로
📝 `range(0,10,2)` → 0,2,4,6,8
🎯 함정: step음수면 역순
⚠️ range(5,0,-1) → 5,4,3,2,1
🔄 `range(10,0,-2)` → 10,8,6,4,2

## [34] while문 기본
📚 while 조건: 조건True인동안반복
📝 `____ x<5: print(x); x+=1` → while
🎯 함정: 무한루프 주의
⚠️ 조건 언젠간 False되어야
🔄 `x=0; while x<3: print(x); x+=1` → 0 1 2

## [35] break문
📚 break=반복문즉시탈출
📝 `for i in range(10): if i==5: ____` → break
🎯 함정: break는 가장가까운반복문만탈출
⚠️ 중첩반복문에서 바깥탈출불가
🔄 `while True: break` → 1회후종료

## [36] continue문
📚 continue=현재반복건너뛰고다음반복
📝 `if i%2==0: ____` 짝수건너뛰기 → continue
🎯 함정: continue vs break vs pass
⚠️ continue는 반복문안에서만
🔄 `for i in range(5): if i==2: continue; print(i)` → 0 1 3 4

## [37] 중첩 for문
📚 for안에for, 바깥1회당안쪽전체
📝 구구단=중첩for문
🎯 함정: 변수명 다르게 (i,j)
⚠️ 바깥break→안쪽영향없음
🔄 `for i in range(2): for j in range(3): print(i,j)`

## [38] enumerate()
📚 enumerate=인덱스+값 동시반환
📝 `for i,v in ____(리스트):` → enumerate
🎯 함정: i,v 순서 (인덱스,값)
⚠️ enumerate(리스트,시작값)
🔄 `for i,c in enumerate("ab"): print(i,c)` → 0 a, 1 b

## [39] 반복+조건 조합
📚 for문안에if문, 조건필터링
📝 짝수만출력: `if i%2==0: print(i)`
🎯 함정: 들여쓰기레벨
⚠️ for→if→실행문 3단계들여쓰기
🔄 `for i in range(10): if i%2==0: print(i)`

## [40] 반복문 else
📚 for-else: break없이완료시else실행
📝 `for i in range(5): ... ____: print("완료")` → else
🎯 함정: break시 else 실행안됨
⚠️ 잘 안쓰이지만 출제가능
🔄 `for i in range(3): pass else: print("끝")` → 끝

## [41] 리스트 기초
📚 리스트=여러값저장, [값1,값2,...], 순서있음
📝 `a = ____1, 2, 3]` → [
🎯 함정: []대괄호, ()소괄호 구분
⚠️ 빈리스트=[] 또는 list()
🔄 `[1,2,3], ["a","b"], [1,"a",True]`

## [42] 리스트 인덱싱
📚 리스트[인덱스], 0부터시작, 음수=뒤에서
📝 `a=[1,2,3]; a[0]` → 1
🎯 함정: a[1]은 두번째요소!
⚠️ a[-1]=마지막, a[-2]=끝에서두번째
🔄 `a=[10,20,30]; a[0],a[-1]` → 10, 30

## [43] 리스트 슬라이싱
📚 리스트[시작:끝], 끝미포함, [:]=전체복사
📝 `a=[1,2,3,4,5]; a[1:4]` → [2,3,4]
🎯 함정: 끝인덱스미포함!
⚠️ a[:3]=[1,2,3], a[2:]=[3,4,5]
🔄 `a[::2]` → [1,3,5] (2칸씩)

## [44] 리스트 수정
📚 리스트[인덱스]=새값, 직접수정
📝 `a=[1,2,3]; a[0]=10; a` → [10,2,3]
🎯 함정: 범위초과시 오류
⚠️ a[10]=5 → IndexError
🔄 `a[1:3]=[20,30]` → 슬라이스수정가능

## [45] 리스트 append()
📚 append(값)=리스트끝에추가
📝 `a=[1,2]; a.____(3)` → append
🎯 함정: append는 None반환
⚠️ a=a.append(3) → a가 None됨!
🔄 `a=[]; a.append(1); a.append(2)` → [1,2]

## [46] 리스트 insert()
📚 insert(인덱스,값)=특정위치에삽입
📝 `a=[1,3]; a.insert(1,2); a` → [1,2,3]
🎯 함정: insert도 None반환
⚠️ 기존요소는 뒤로밀림
🔄 `a=[1,2,3]; a.insert(0,0)` → [0,1,2,3]

## [47] 리스트 remove(), pop()
📚 remove(값)=값삭제, pop(인덱스)=인덱스삭제후반환
📝 `a=[1,2,3]; a.remove(2); a` → [1,3]
🎯 함정: remove는 첫번째만 삭제
⚠️ pop()=마지막삭제, pop(0)=첫번째삭제
🔄 `a=[1,2,3]; a.pop()` → 3반환, a=[1,2]

## [48] 리스트 정렬
📚 sort()=정렬(원본변경), sorted()=정렬(새리스트)
📝 `a=[3,1,2]; a.sort(); a` → [1,2,3]
🎯 함정: sort()는 None반환!
⚠️ b=a.sort() → b는 None
🔄 `sorted([3,1,2])` → [1,2,3] (원본유지)

## [49] 리스트 함수들

📝 `len([1,2,3])` → 3
🎯 함정: sum은 숫자리스트만
⚠️ sum(["a","b"]) → TypeError
🔄 `min([3,1,2]), max([3,1,2])` → 1, 3

## [50] 리스트 in 연산자

🎯 함정: ==와 in 혼동
⚠️ [3] in [1,2,3] → False (3≠[3])
🔄 `"a" in ["a","b"]` → True

## [51] 문자열 기초
📚 문자열=""또는'', 불변(immutable)

🎯 함정: 문자열 직접수정불가

🔄 `s="hi"; s=s+"!"` → "hi!" (새문자열)

## [52] 문자열 인덱싱
📚 문자열[인덱스], 0부터, 음수=뒤에서

## [53] 문자열 슬라이싱

## [54] 문자열 연결/반복
📚 +연결, *반복
📝 `"a"+"b"` → "ab"

## [55] 문자열 split()
📚 split(구분자)=문자열→리스트

🎯 함정: split()=공백기준
⚠️ "a  b".split()=["a","b"] (연속공백하나로)
🔄 `"hello world".split()` → ["hello","world"]

## [56] 문자열 join()
📚 구분자.join(리스트)=리스트→문자열
📝 `"-".join(["a","b","c"])` → "a-b-c"
🎯 함정: join은 문자열리스트만
⚠️ ",".join([1,2,3]) → TypeError
🔄 `"".join(["a","b"])` → "ab"

## [57] 문자열 replace()

🎯 함정: 모든 l이 L로 변경
⚠️ replace(a,b,1)=첫번째만변경
🔄 `"aaa".replace("a","b",2)` → "bba"

## [58] 문자열 strip()
📚 strip()=양쪽공백제거, lstrip/rstrip
📝 `"  hello  ".strip()` → "hello"
🎯 함정: 중간공백은 유지
⚠️ " a b ".strip()="a b"
🔄 `"hello".strip("ho")` → "ell"

## [59] 문자열 upper/lower
📚 upper()=대문자, lower()=소문자

🎯 함정: 원본불변, 새문자열반환
⚠️ s.upper() 후 s는 그대로
🔄 `"HeLLo".lower()` → "hello"

🎯 함정: find없으면-1, index없으면에러

🔄 `"hello".find("l")` → 2 (첫번째위치)
