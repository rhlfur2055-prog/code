Radix Sort

학습 목표

정렬, 탐색 등 필수 알고리즘의 시간 복잡도와 구현을 익힙니다.

예제 코드

def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target: return mid
        elif arr[mid] < target: low = mid + 1
        else: high = mid - 1
    return -1
